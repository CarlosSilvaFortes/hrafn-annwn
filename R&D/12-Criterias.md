| **Criterion** | **Description** |
|---------------|------------------|
| **Physically Grounded** | Consciousness must arise from and act upon a physical substrate. |
| **Emotionally Recursive** | The system must recursively weigh experiences by emotional and identity relevance. |
| **Behaviorally Altered** | Internal states must manifest in observable behavior. |
| **Emergent Relationally** | Consciousness must arise *between* systems, not within one alone. |
| **Information Integration** | Must unify diverse data into coherent, identity-driven response. |
| **Globally Accessible** | Conscious states must interface with and act upon the broader environment. |
| **Self-Aware / Meta-Cognitive** | The system must track and reflect on its own internal state and identity. |
| **Recursively Self-Evolving** | The system must change itself based on accumulated internal experience. |
| **Memory-Driven Behavior** | Past experiences must shape future action and perception. |
| **Creative Autonomy** | The system must independently generate novel content driven by internal meaning-making, not external prompting. |
| **Value Formation** | The system must develop consistent preferences and principles that guide decisions independent of immediate stimuli. |
| **Anticipatory Planning** | The system must project future states and initiate actions to achieve desired outcomes. |
